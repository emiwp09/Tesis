# Predicción de Parto Pretérmino mediante Aprendizaje Automático

Este repositorio contiene el código, los conjuntos de datos procesados y los experimentos computacionales asociados a la tesis de licenciatura:

**“Predicción de Parto Pretérmino mediante Modelos de Aprendizaje Automático”**

El objetivo del trabajo es evaluar distintos enfoques de aprendizaje supervisado para la identificación temprana del riesgo de parto pretérmino, a partir de información clínica, metadatos maternos y variables derivadas del microbioma vaginal.
